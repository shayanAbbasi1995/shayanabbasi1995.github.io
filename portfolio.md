## Econometrics & Finance

**[Non-Financial Misconduct and Market Reaction](https://github.com/shayanAbbasi1995/Non-financial-misconduct-and-Market-reaction)**
BSE Master's thesis. Event study on the stock-market reaction to non-financial misconduct by executives. Implements Cumulative Abnormal Return (CAR) models, controls for overlapping event windows, and computes market-quality statistics across 10 simulation sessions. Python / Jupyter.

**[UAB MAREB Thesis](https://github.com/shayanAbbasi1995/UAB-Thesis)**
Empirical thesis for the MAREB program at UAB on market power, regulation, and firm-level investment. Panel regressions (OLS, FE, two-way FE) on Compustat data, robustness checks with alternative markup measures, and SEC EDGAR data scraping for litigation matching. Stata / Python.

**[BGSE Macro-Finance](https://github.com/shayanAbbasi1995/BGSE-Macro-Finance)**
Four R assignments on asset pricing: equity premium puzzle (CCAPM calibration), Hansen-Jagannathan bounds, habit formation, and the term structure of interest rates. Uses Jordà–Schularick–Taylor historical macroeconomic data. R / R Markdown.

**[UAB MAREB Assignments](https://github.com/shayanAbbasi1995/UAB-MAREB-Assignments)**
Course assignments covering econometrics, multivariate analysis (clustering, factor analysis, discriminant analysis, panel data), applied microeconomics, applied macroeconomics, production efficiency, and finance. Stata / R.

## Machine Learning & NLP

**[BGSE Text Mining](https://github.com/shayanAbbasi1995/bgse-text-mining-assignments)**
Four assignments progressing from text preprocessing and TF-IDF through document classification (Naive Bayes, logistic regression) and Word2Vec embeddings to a final end-to-end NLP pipeline on a real-world dataset. Python / Jupyter.

**[ML Classification — ICU Mortality](https://github.com/shayanAbbasi1995/bgse-ml-classification)**
Binary classification project predicting in-hospital mortality using the MIMIC-III critical care database (40,000+ ICU patients). Covers feature engineering, model comparison, and evaluation. Python / Jupyter.

**[ML for Finance — Trading Strategy Backtesting](https://github.com/shayanAbbasi1995/bgse-ml-finance-backtesting)**
Portfolio optimization and back-testing assignment using AMZN price data with NLP-derived sentiment signals (positive, negative, certainty, financial up/down scores). Implements mean-variance optimization and evaluates strategy performance over a 2018–2019 out-of-sample window. R / R Markdown.

## Quantitative Finance & Trading

**[Crypto Grid Trading Backtest](https://github.com/shayanAbbasi1995/crypto-grid-trading-backtest)**
Python backtesting framework for a grid trading strategy on crypto futures. Reconstructs position holding periods from raw trade exports, computes per-trade net returns after fees using geometric averaging, and generates PnL distribution plots and VaR estimates. Python.

## Web Scraping & Data Collection

**[APKMirror Crawler](https://github.com/shayanAbbasi1995/apkmirror-crawler)**
Production-grade Selenium scraper for APKMirror. Collects structured APK metadata (title, developer, category, permissions, languages, size, OS requirements) at scale with automatic Cloudflare challenge handling, fullscreen ad dismissal, DMCA detection, and resumable crawling via a visited-links log. Python.

**[Codal Financial Scraper](https://github.com/shayanAbbasi1995/codal-financial-scraper)**
Selenium scraper for Codal.ir, the official disclosure portal of the Iranian stock exchange. Downloads and parses monthly activity reports and annual/interim fiscal statements (balance sheets, income statements, cash flow) for listed companies. Python.

**[Web Scraping & API Samples](https://github.com/shayanAbbasi1995/web-scraping-and-api-samples)**
Two notebooks: (1) scraping SEC EDGAR for 10-K filings and (2) Twitter API data collection with sentiment analysis on ICO-related tweets. Python / Jupyter.
